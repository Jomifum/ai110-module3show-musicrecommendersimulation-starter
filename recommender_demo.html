VibeFinder 1.0 — interactive demo

A content-based music recommender simulation.

This page is a browser companion to the CLI tool. It reimplements the same scoring recipe used in src/recommender.py (genre match +2.0, mood match +1.0, energy similarity up to +1.0) in JavaScript so you can explore it interactively. The Python CLI remains the primary deliverable. Tip: set genre to "(none)" to see how the genre anchor controls which songs surface.
